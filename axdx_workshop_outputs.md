# Problem Definition

## 1. 주제
AI 기반 스타일 솔루션

## 2. 문제 상황
자신의 체형과 취향에 딱 맞는 옷을 찾는 과정에서 겪는 '선택의 어려움'과 '구매 실패'
- 정보 과잉으로 인한 선택 피로
- 모델 핏과 실제 착용 핏의 괴리
- 개인화된 코디 가이드의 부재
- 입력의 번거로움

## 3. 핵심 사용자
패션 정보의 홍수 속에서도 나만의 정답을 찾지 못하는 소비자들

## 4. 사용자가 겪는 불편
모델 사진만 믿고 샀다가, 실망하며 반품한다.

## 5. 현재 대안과 한계
스타일 추천 서비스가 제공되지만, AI 기반의 추천이 개인의 세밀한 감정적/상황적 니즈를 모두 충족시키지 못하는 한계 존재 (개인화 추천의 한계)

## 6. 서비스 아이디어
기존의 번거로운 수치 입력을 대신하여 Vision AI와 인터렉티브 카드를 통해 사용자의 스타일 DNA를 분석한다. 
- Body DNA 스캔
- 취향 데이터 라벨링
- 에이전트 코디 매칭

## 7. 4주 안에 구현 가능한 최소 기능
비전 기반 간이 체형 분석, 인터렉티브 스타일 스와이프, 체형 맞춤형 큐레이션, 기본 마이페이지


## 8. 데이터로 검증할 수 있는 지표
분석 완료율, 스타일 선호도 데이터 밀도, 추천 코디 클릭률, 공유 및 저장 수 

## 9. 구현 범위에서 제외할 것
직접적인 커머스 결제 시스템, 정밀 수치 측정, 실시간 가상 피팅, 커뮤니티 및 소셜 기능, 정교한 백엔드 어드민

# Persona

- 이름: 최현아
- 나이: 24
- 전공: 경영학과
- 학년: 마케팅팀 신입사원

## 하루 일과
점심시간을 활용해 습관적으로 패션 앱을 탐색하며, 출근 룩과 주말 데이트 룩을 고민함.

## 자주 사용하는 도구
지그재그, 무신사, 네이버쇼핑

## 목표
출근 및 데이트 시 실패없는 룩을 최단 시간에 완성하는 것


## Pain Point
모델 사진만 믿고 구매했다가 실제 체형 차이로 인해 옷이 어울리지 않아 반복적으로 반품함. 
- 수많은 상품을 직접 스크롤하며 코디를 조합하는 과정에서 과도한 에너지 소모와 선택 피로감을 느낌
- 상세 페이지의 수치만으로는 실제 착용 핏을 예측하기 어려움


## 서비스 사용 전 행동
패션 앱의 후기를 샅샅이 뒤져 자신과 비슷한 키와 몸무게를 가진 사람을 찾느라 귀한 휴식 시간을 낭비함

## 서비스 사용 후 기대 변화
- 개인 취향에 맞는 스타일 쉽게 발견 가능
- 빠르고 만족도 높은 코디 선택 경험 제공


## 절대 원하지 않는 것
복잡한 설정, 불필요한 단계 (복잡한 회원가입 절차)

# PRD

## 문제정의
자신의 체형과 취향에 딱 맞는 옷을 찾는 과정에서 겪는 '선택의 어려움'과 '구매 실패'
- 정보 과잉으로 인한 선택 피로
- 모델 핏과 실제 착용 핏의 괴리
- 개인화된 코디 가이드의 부재
- 입력의 번거로움

## 목표 사용자
패션 정보의 홍수 속에서도 나만의 정답을 찾지 못하는 소비자들

## 핵심 가치 제안
모델의 외형에 의존한 쇼핑 환경에서 벗어나, 사용자의 객관적 신체 데이터와 시각적 취향 라벨링을 결합하여 쇼핑 실패율과 탐색 피로도를 획기적으로 낮춥시다.


## 핵심 시나리오
1. 진단: 사용자가 전신 사진을 촬영하면 Vision  AI가 체형을 정밀 분석
2. 구축: 텍스트 대신 시각적인 코디 카드를 선책하며 개인의 스타일 취향 데이터를 구축
3. 매칭: 분석된 체형 DNA아 취향 데이터를 매칭하여 에이전트가 자동 코디 세트 제안
4. 탐색: 제안된 코디를 피드 형태로 탐색하며 마음에 드는 스타일을 발견하고 저장

## MVP 기능 범위
1. 비전 기반 Body DNA 스캔
2. 인터렉티브 스타일 라벨링
3. 스타일 에이전트 추천 피드
4. 스타일 보관함 (찜)

## 제외할 기능
직접적인 커머스 결제 시스템, 정밀 수치 측정, 실시간 가상 피팅, 커뮤니티 및 소셜 기능, 정교한 백엔드 어드민

## 주요 데이터
Physical Data, Preference Data, Product Data

## KPI
- 추천 코디 구매 전환율 (CVR): (에이전트 추천 코디를 통한 구매 건수 / 추천 코디 총 노출 수) × 100 / 목표 기존 패션몰 평균 대비 3배 이상인 4.5% 달성
- 제안 상품 반품률 (Return Rate): (StyleDNA 추천 상품 중 반품 건수 / 전체 구매 및 반품 건수) × 100 / 목표 기존 대비 50% 이상 감소한 10% 이하 유지
- 월간 활성 사용자 재방문율 (Retention Rate): (이번 달에 다시 방문한 기존 유저 수 / 지난달 전체 활성 유저 수) × 100 / 목표 유닛 이코노믹스 기준인 월 이탈률 2% 방어 (재방문율 98%)

## 리스크
기술적 의존도, 데이터 콜드 스타트, 개인정보

## 향후 확장 방향
커머스 연동, B2B 데이터 사업, 가상 피팅 (AR)

# Acceptance Criteria

| ID | 기능 | Given | When | Then | 우선순위 |
|---|---|---|---|---|---|
| 비전 AI 기반 Body DNA 스캔 | 전신 사진 분석을 통한 체형 데사용이터 추출 및 데이터 추출 및 리포트 생성 | 사용자가 앱 내의 'Body DNA 스캔' 카메라 화면에서 전신이 모두 보이도록 서 있는 상태에서 | 촬영 버튼을 누르거나 사진을 업로드하면 | Vision AI 엔진이 얼굴형, 어깨, 허리, 골반 라인을 정밀 분석하여 사용자의 3대 체형(스트레이트, 웨이브, 내추럴) 유형과 정량적 측정 데이터(어깨 너비, 허리 둘레 등)가 포함된 리포트 페이지를 화면에 명확히 보여준다. | High |
| 인터랙티브 스타일 라벨링 | 시각적 코디 카드를 통한 4단계 취향 데이터 구축 | 사용자가 체형 진단을 마치고 '취향 설정하고 코디 추천받기' 버튼을 선택하여 라벨링 단계에 진입했을 때 | 총 4단계에 걸쳐 텍스트 입력 없이 화면에 제시되는 시각적인 '스타일(미니멀, 캐주얼 등)', '색상', '상황(TPO)', '마음에 드는 룩' 카드를 터치하여 다중 선택하고 '다음' 버튼을 누르면 | 시스템은 유저가 선택한 시각적 속성값들을 실시간으로 라벨링 데이터로 누적하고, 상단의 인디케이터(예: 1/4, 2/4)를 통해 진행 상황을 직관적으로 표시한다. | High |
| 에이전트 자동 코디 매칭 | 체형 리포트와 취향 데이터를 결합한 커스텀 스타일 추천 | 사용자가 4단계의 시각적 스타일 라벨링을 모두 완료하고 '완료하고 시작하기' 버튼을 누르면 | 스타일 매칭 에이전트 알고리즘이 가동되어 사용자의 신체 데이터와 취향 라벨을 결합 분석할 때 | 사용자의 체형적 단점은 보완하고 선호 스타일에 부합하는 상·하의 및 액세서리가 조합된 자동 코디 세트(예: 역삼각형 체형을 위한 슬림핏 재킷 + 테이퍼드 팬츠 조합)를 즉시 매칭하여 메인 화면에 출력한다. | High |
| 인터랙티브 스타일 탐색 피드 (찜 기능) | 추천 코디 탐색 및 마음에 드는 스타일 저장 | 사용자가 개인화 매칭이 완료된 '스타일 탐색' 무한 스크롤 피드 화면에 머물러 있을 때 | 피드를 훑어보며 마음에 드는 코디 카드의 '하트(좋아요)' 아이콘을 터치하거나 상세 아이템을 클릭하면 | 하트 아이콘이 활성화 상태로 변경되며 해당 코디 세트가 사용자의 보관함(찜 리스트)에 즉시 저장되고, 유저가 선택(클릭/좋아요)을 반복할수록 추천 엔진의 분석 범위가 좁혀지며 다음 피드 로딩 시 더 정교한 스타일 위주로 자동 재배치된다. | Medium |

# KPI

|---|---|---|
| 추천 코디 구매 전환율 (CVR) | (에이전트 추천 코디를 통한 구매 건수 / 추천 코디 총 노출 수) × 100 | 기존 패션몰 평균 대비 3배 이상인 4.5% 달성 |

# Experiment Plan

## 1. 실험 가설
기존의 복잡한 텍스트 검색이나 필터링 방식 대신, 시각적인 스타일 카드를 넘기는(스와이프) 인터랙티브 UX를 제공하면, 유저들의 탐색 피로도가 줄어들어 스타일 분석 완료율이 20% 이상 증가할 것이다."

## 2. 실험군과 대조군
- Control: 스타일 선호도를 파악하기 위해 미니멀, 캐주얼 등의 스타일 단어를 텍스트 목록(체크박스)으로 선택하게 하는 화면
- Variant: 3팀의 핵심 아이디어인 시각적인 코디 이미지 카드를 좌우로 스와이프하며 직관적으로 취향을 라벨링하는 화면

## 3. 주요 지표
 분석 완료율 (첫 진입 유저 중 최종 취향 분석을 완료한 유저 비율)

유저당 평균 취향 데이터 라벨링(선택) 개수

## 4. 성공 기준
실험군(Variant)의 분석 완료율이 대조군(Control) 대비 통계적으추가한다.로 유의미하게 20% 이상 높게 나타날 경우 성공으로 판단

## 5. 데이터 수집
각 화면 진입 및 이탈 로그 데이터 수집

유저별 카드 스와이프/체크박스 클릭 횟수 및 체류 시간 수집 (Amplitude 또는 Google Analytics 활용)
